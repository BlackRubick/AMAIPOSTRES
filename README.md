# Amai Postres - Landing (Vite + React + Tailwind)

Proyecto de ejemplo: landing page premium para "Amai Postres Repostería".

Tecnologías:
- React + Vite
- Tailwind CSS
- Framer Motion (animaciones)
- Lucide React (iconos)

Requisitos:
- Node.js 18+ recomendado

Instalación:

```bash
# en la carpeta del proyecto
npm install
# iniciar en desarrollo
npm run dev
```

Notas rápidas:
- El botón de WhatsApp abre https://wa.me con el número +51 9613037073 (ajusta si necesitas otro código de país).
- Los productos están en `src/data/products.js` y usan imágenes de placeholder (Unsplash). Reemplaza por tus fotos.
- Configura theme y colores en `tailwind.config.cjs`.

Siguientes pasos sugeridos:
- Añadir optimización de imágenes y contenedor `public/` con assets locales.
- Integrar un CMS o archivo de traducciones si se requiere multi-idioma.
- Añadir formularios y validaciones para pedidos personalizados.
